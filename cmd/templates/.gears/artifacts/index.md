## Adding an Artifact

1. Create a `.md` file in this directory (e.g., `controller-pattern.md`, `auth-flow.md`)
2. Add a row to the index table above
3. Reference it from `.gears/instructions/index.md` if it defines a reusable pattern

Artifact files can contain: code examples, annotated snippets, schema definitions, config templates, or architecture diagrams in text/Mermaid.
