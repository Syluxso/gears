# Architectural Decision Records

One entry per significant decision. **Never edit existing entries** — add new ones to supersede old ones. Decisions are permanent history.

---

## ADR Format

```markdown
## ADR-XXX: [Title]

**Date:** YYYY-MM-DD
**Status:** Accepted | Superseded by ADR-XXX

### Context

What situation prompted this decision?

### Decision

What did we decide?

### Options Considered

- Option A — pros/cons
- Option B — pros/cons

### Reasoning

Why did we choose this option over the others?

### Consequences

What does this change or constrain going forward?
```
